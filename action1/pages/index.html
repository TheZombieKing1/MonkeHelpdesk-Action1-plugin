Configuration

Found in Action1 → Settings → API Keys → Client ID.

Write-only. Leave blank to keep existing value.

Enter Client ID + Secret above first, then click Auto-detect.

Required Action1 API permissions: In Action1 → Settings → API Keys, the credential must have a role that includes Computers / Endpoints → Read. The built-in Viewer role is sufficient. Without it, syncs return 403.

Comma-separated. Group names with spaces work fine (e.g. Sales Laptops, Finance PCs).
